## 项目内容

### CSS长度单位探索器
一个科技风格的交互式CSS长度单位可视化工具，具有以下特性：

#### 主要功能
- **单位反应堆**：实时演示各种CSS长度单位的效果
- **时空对比仪**：比较不同单位之间的数值关系
- **单位图鉴**：展示所有CSS长度单位的详细信息和可视化表示
- **参数控制**：可调节基准字体大小和容器宽度来观察单位变化

#### 支持的单位
- **绝对单位**：px, cm, mm, Q, in, pc, pt
- **相对单位**：em, rem, ex, ch, vw, vh, vmin, vmax, %, cap, ic, lh, rlh, vi, vb

#### 技术特色
- 响应式设计，适配不同屏幕尺寸
- 科技风格的霓虹色彩主题
- 丰富的动画和过渡效果
- 实时计算和可视化展示
